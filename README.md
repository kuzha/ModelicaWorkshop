# Modelica Workshop
This repository contains Modelica training materials. 

## Modelica Buildings Library (MBL)
This workshop uses the MBL in the tutorials. You can download the library from [link](https://simulationresearch.lbl.gov/modelica/download.html).

## OpenModelica simulation environment
If you don't have a licence for commercial software (e.g., Dymola), you can download the open-source software [OpenModelica](https://openmodelica.org/) to edit and run Modelica models.
